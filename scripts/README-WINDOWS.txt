========================================
Kyma for Windows
========================================

Quick Start:
1. Extract Kyma-Windows-x64.zip
2. Double-click run-kyma.bat
3. Or run kyma.exe directly

Requirements:
- Windows 10 or later
- WebView2 Runtime (usually pre-installed)

Troubleshooting:
- If you see "mpv.exe not found", you extracted incorrectly
- Make sure all files are in the SAME folder
- Run from the extracted folder, not from inside the zip

For YouTube/SoundCloud support:
- All dependencies (mpv, yt-dlp, ffmpeg) are included
- No additional installation needed
